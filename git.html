INTRODUCTION TO GIT

GIT VERSIONS

GIT first version 0.99 was released on 11 JULY 2005.
GIT current version 2.13 was released on 10 MAY 2017.

Some comman commands of GIT are:

  1. git status : Show the working tree status.
  2. git add : Add file contents to the index.
  3. git commit : Record changes to the repository.
  4. git push : Update remote refs along with associated objects.
  5. git mv : Move or rename a file or a directory.
  6. git clone : Clone a repository into a new directory.
  7. git grep : Print lines matching a pattern.
  8. git log : Show commit logs.
  9. git diff : Show changes between commits, commit and working tree etc.
  10. git merge : Join two or more development histories together
