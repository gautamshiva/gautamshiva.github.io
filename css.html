INTRODUCTION TO CSS

CSS stands for Cascading Style Sheets.

VERSIONS

CSS first version was released in December 17, 1996.
CSS current version was released on 2017.

Some common CSS commands are:

  1. background-color: specifies the background color of an element.
  2. border-style : specifies what kind of border to display.
  3. margin-bottom :sets the bottom margin of an element.
  4. max-width : is used to set the maximum width of an element.
  5. text-align : is used to set the horizontal alignment of a text.
  6. max-height :Sets the maximum height of an element.
  7. text-align :Specifies the horizontal alignment of text.
  8. vertical-align :Sets the vertical alignment of an element.
  9. font-size :Specifies the font size of text.
  10. font-style :Specifies the font style for text.
