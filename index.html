HTML stands for hyper text markup language
VERSIONS
HTML first version was released in 1993.
HTML current version was released on 28 October 2014.
Some comman commands of HTML are:
1.h1 to h6 - h1 defines the most important heading and h6 defines the least important heading.
2.p - defines paragraphs.
3.a - defines link.
4.ul - defines the unordered list.
5.ol - defines the ordered list.
